## Getting Started

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file and add your Telegram Bot Token:

```bash
BOT_TOKEN=your_bot_token_here
```

4. Run the bot:

```bash
node app.js
```
